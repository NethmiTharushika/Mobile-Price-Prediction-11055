# Mobile-Price-Prediction-11055

This project uses machine learning techniques to predict the price range of mobile phones based on features such as RAM, battery power, and screen size.

# Algorithms Used
- Logistic Regression
- Decision Tree
- K-Nearest Neighbors (KNN)

# Results
KNN achieved the highest accuracy of 94.25%, making it the best model for this dataset.
